# Big Message Remote Control

The display page reads from `settings.json` every 5 seconds.

To change the words remotely, update `settings.json` and publish that file with the page.

Settings:

- `message`: the big text on screen
- `label`: the small badge in the corner
- `theme`: `sunset`, `midnight`, `mint`, or `rose`
- `font`: `sans`, `serif`, or `script`
- `size`: a CSS size like `12vw`
- `uppercase`: `true` or `false`
- `outline`: `true` or `false`
- `soft`: `true` or `false`
